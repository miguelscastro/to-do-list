# To-do List

A React to-do list application that demonstrates tasks and state management.

---

## 🔧 Prerequisites

- Node.js 16+ installed

---

## 🚀 Quick start (local)

```bash
cd reactjs/01-todo-list
npm install
npm run dev
```

---

## 🧪 Tests

- No automated tests included.

---

## 🔁 Features (summary)

- Task creation and completion
- Filtering by status
